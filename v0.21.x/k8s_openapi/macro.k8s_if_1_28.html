Macro k8s_openapi::k8s_if_1_28

macro_rules! k8s_if_1_28 {

This macro evaluates to its contents if the v1_28 feature is enabled, otherwise it evaluates to nothing.

§Examples

k8s_if_1_28! {
    use k8s_openapi::api::core::v1 as api;
}
